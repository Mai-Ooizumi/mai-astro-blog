# Hello, World! 你好，世界！

## Technologies 本站点使用的技术栈

- [**Astro**](https://astro.build/) : 一个现代化的静态站点生成器，支持多种前端框架和组件。

- [**Astro Assets**](https://docs.astro.build/en/guides/assets/) : Astro的资源管理功能，用于处理图像、字体等静态资源。

- [**Astro Icon**](https://github.com/natemoo-re/astro-icon) : Astro的图标组件库，提供多种图标选择和使用方式。

- [**Astro-Pagefind**](https://github.com/shishkin/astro-pagefind) : Astro的搜索引擎集成，提供快速和高效的站内搜索功能。

- [**Markdown**](https://www.markdownguide.org/) : 一种轻量级标记语言，用于格式化文本内容。

- [**MDX**](https://mdxjs.com/) : Markdown与JSX的结合，允许在Markdown中使用其他语言的组件。

- [**Tailwind CSS**](https://tailwindcss.com/) : 实用优先的CSS框架，用于快速构建自定义设计。
